# 更新日誌 (Changelog)

本專案遵循 [語意化版本](https://semver.org/lang/zh-TW/) 規範。

## [1.2.0] - 2026-01-19

### 圖片瀏覽體驗升級

#### Lightbox 全螢幕檢視器
- 點擊食譜圖片開啟全螢幕檢視模式
- 支援放大/縮小功能（點擊圖片或使用按鈕）
- 鍵盤快捷鍵支援（ESC 關閉、左右箭頭切換）
- 黑色背景突顯圖片細節

#### 圖片滑動動畫
- 圖片切換時加入平滑過渡動畫效果
- 淡入淡出搭配滑動位移
- 更流暢的視覺體驗

#### 手勢操作優化
- Lightbox 支援觸控滑動切換圖片
- 分頁指示器支援動畫效果
- 點擊背景關閉 Lightbox

### 技術更新
- 新增 ImageLightbox 元件
- 優化動畫效能使用 CSS transitions

## [1.1.0] - 2026-01-19

### 效能優化

#### 圖片縮圖機制
- 新增 `/thumbnails/` API 端點，動態產生壓縮縮圖
- 首頁食譜卡片使用 400x300 縮圖，大幅加速載入速度
- 清單模式使用 80x80 小縮圖，進一步優化效能
- 縮圖快取一年，減少重複處理

### 新增功能

#### 檢視模式切換
- 新增大圖模式與清單模式切換按鈕
- 清單模式可一次瀏覽更多食譜
- 保留原有的分類篩選和搜尋功能

### 技術更新
- 整合 Sharp 圖片處理庫進行縮圖壓縮
- 新增 RecipeListItem 元件用於清單顯示

## [1.0.0] - 2026-01-11

### 新增功能

#### 多圖片支援
- 每個食譜支援上傳最多 5 張照片
- 使用 presigned URL 直接上傳至雲端儲存
- 支援 JPG、PNG、WebP 格式，每張最大 10MB
- 使用唯一 ID 追蹤機制解決多檔案同時上傳的競態條件問題

#### 圖片輪播與手勢操作
- 食譜詳情頁支援圖片輪播功能
- 左右箭頭按鈕切換圖片
- 點狀指示器顯示當前圖片位置
- 手機觸控滑動手勢支援（左右滑動切換圖片）

#### 多來源網址
- 每個食譜可新增多個來源網址（影片、部落格連結等）
- 動態新增/移除網址輸入欄位
- 食譜詳情頁顯示所有來源連結

#### 編輯功能增強
- 編輯時可個別刪除食譜照片
- 紅色圓形刪除按鈕，在手機上更易操作
- 支援新增照片至現有食譜

### 資料庫架構

- 使用 PostgreSQL 資料庫持久化儲存
- Drizzle ORM 提供型別安全的資料庫操作
- 整合 Replit Object Storage 雲端圖片儲存

### 技術規格

- **前端**: React 18 + TypeScript + Vite
- **後端**: Node.js + Express + TypeScript
- **樣式**: Tailwind CSS + shadcn/ui
- **資料庫**: PostgreSQL + Drizzle ORM
- **檔案儲存**: Replit Object Storage (GCS)

### 介面設計

- 手機優先的響應式設計
- 繁體中文介面
- 溫暖親密的配色方案
- 觸控優化的操作介面
